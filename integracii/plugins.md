# Веб-расширения

{% hint style="info" %}
Подключение расширений доступно только в серверной версии Бипиума.
{% endhint %}

## Создание расширений

### 1. Выберите уникальный код

Каждое расширение имеет уникальный код-идентификатор. Код может состоять из латинских букв, цифр и дефиса.

### 2. Напишите расширение

Исходный код минимального расширения доступен в публичном репозитории: [https://bitbucket.org/snippets/bpium/d4bMe/blank-modal-extension](https://bitbucket.org/snippets/bpium/d4bMe/blank-modal-extension)<br>

### 3. Зарегистрируйте расширение

Бипиум защищает расширения от тиражирования через систему лицензирования. Подключение расширения к системе возможно только при наличии лицензии.

Чтобы зарегистрировать расширение — напишите на **support@bpium.ru**, указав уникальный код расширения и регистрационный номер лицензии компании.

{% hint style="info" %}
Без регистрации расширения в системе лицензирования подключить его будет невозможно.
{% endhint %}

## Подключение расширений

1. Разместите файлы расширения на внешнем веб-сервере или в рабочей папке сервера Бипиума:\
   `bpium_path/www/modules/my-extension/`\
   Если серверов несколько — скопируйте файлы на каждый.
2. Откройте каталог «Расширения» в разделе «Система».
3. Создайте новую запись со следующими полями:

<table><thead><tr><th width="157.6363525390625">Поле</th><th>Описание</th></tr></thead><tbody><tr><td><strong>Название</strong></td><td>Название расширения в главном меню приложения</td></tr><tr><td><strong>Иконка</strong></td><td>Иконка в главном меню</td></tr><tr><td><strong>Код</strong></td><td>Уникальный идентификатор, должен совпадать с переменной <code>moduleName</code> в скрипте</td></tr><tr><td><strong>JS URL</strong></td><td>Путь до JS-скрипта. Например: <code>modules/my-extension/main.js</code></td></tr><tr><td><strong>CSS URL</strong></td><td>Путь до CSS-файла. Например: <code>modules/my-extension/main.css</code></td></tr></tbody></table>

4. Сохраните запись и обновите страницу в браузере. Если всё верно — во вкладке Network браузера появятся запросы к вашим файлам.

#### Проверка подключения

При старте веб-приложение Бипиум делает запрос к серверу `/auth/company`. Ответ содержит массив подключённых расширений:

```
{ "domain": "admin", "title": "Моя компания", "extensions": [ массив расширений ] }
```

Сервер не вернёт расширения, на которые нет лицензии для данной компании.
