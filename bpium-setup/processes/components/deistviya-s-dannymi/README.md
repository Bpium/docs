# Действия с данными

### Когда использовать

Действия с данными нужны в любом сценарии, который обращается к записям в Бипиуме. Типичные примеры:

* При создании заявки — получить данные клиента из другого каталога
* Перед отправкой отчёта — найти все записи за определённый период
* При изменении статуса — обновить связанные записи
* При загрузке файла — сохранить его в нужную запись
* Сформировать договор из шаблона и прикрепить к заявке

### Доступ к данным

**Важно:** Процессы имеют доступ к записям **минуя правовую политику**. Это значит, что сценарий может читать, создавать, изменять и удалять любые записи, даже если у сотрудника, запустившего сценарий, нет на это прав. Используйте эту возможность аккуратно.
