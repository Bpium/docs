# Сервис получения токенов

Для того, чтобы получить токен, откройте tokens.bpium.ru и следуйте инструкции. <br>

<figure><img src="../../../../../.gitbook/assets/image (18).png" alt=""><figcaption></figcaption></figure>

Для примера выберем сервис Телеграм и введем данные API ID и API Hash.

<figure><img src="../../../../../.gitbook/assets/image (19).png" alt=""><figcaption></figcaption></figure>

Чтобы их получить перейдите по адресу [https://my.telegram.org/auth](https://my.telegram.org/auth) и войдите с помощью номера телефона, который хотите привязать в качестве отправителя.

<figure><img src="../../../../../.gitbook/assets/image (13).png" alt=""><figcaption></figcaption></figure>

Перейдите в API development tools

<figure><img src="../../../../../.gitbook/assets/image (14).png" alt=""><figcaption></figcaption></figure>

Создайте новое приложение, заполнив поля

<figure><img src="../../../../../.gitbook/assets/image (15).png" alt=""><figcaption></figcaption></figure>

Заполните поля конфигурации API

<figure><img src="../../../../../.gitbook/assets/image (237).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../../../../.gitbook/assets/image (238).png" alt=""><figcaption></figcaption></figure>

Скопируйте API ID и API hash  и вставьте в поля на странице tokens.bpium.ru&#x20;

<figure><img src="../../../../../.gitbook/assets/image (229).png" alt=""><figcaption></figcaption></figure>

После подтверждения авторизации вы получите ключ.
