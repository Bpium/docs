# Свойства

У поля помимо свойств, содержащихся на панели, есть также индивидуальные свойства, о которых можно прочитать в статье каждого поля. Для выбора статьи конкретного поля перейдите в [Поля.](edit/)

## Общие свойства

#### **Название поля**

Позволяет задать необходимое название полю.

#### **Подсказка**

При заполнении данного поля подсказка будет отображаться сотрудникам в момент заполнении поля в анкете.

#### **Признак обязательного поля**

После активации данного свойства поле становится обязательным для заполнения при создании записи.

<figure><img src="../../.gitbook/assets/image (7) (1) (1).png" alt=""><figcaption><p>Признак обязательного поля.</p></figcaption></figure>



#### **Показывать на панели фильтров**

При активированном свойстве поля можно будет фильтровать записи по значению поля.

<figure><img src="../../.gitbook/assets/image (8) (1).png" alt=""><figcaption><p>Настройка отображения поля в фильтрах.</p></figcaption></figure>



#### **Сохранить изменения в истории**

При активированном свойстве поля изменения данного поля можно будет просмотреть в истории изменений.

<figure><img src="../../.gitbook/assets/image (9) (1).png" alt=""><figcaption><p>Сохранение изменений поля в истории.</p></figcaption></figure>



#### **Настройка вида отображения**

Если задать ограничения для редактирования или отображения поля через настройку вида отображения, то ни у одного пользователя (в том числе администратора) не будет возможности расширить права доступа к полю через права доступа к каталогу.&#x20;

#### **Виды отображения**

**Вид «Редактируемое» позволяет редактировать значение поля.** Когда для поля выбран вид отображения Редактировать, поле отображается в карточке записи и его можно редактировать из карточки записи. Но поле может быть изменено через автоматизации.

**Вид «Просмотр» позволяет только видеть значение поля.** Когда для поля выбран вид отображения Видеть, поле отображается в карточке записи, но его нельзя редактировать из карточки записи вручную. При этом поле может быть изменено от имени пользователя автоматизациями, запускаемыми во время редактирования карточки или через АПИ.

**Вид «Скрыто» скрывает поле со значением.** Когда для поля выбран вид отображения Скрыто, поле не отображается в карточке записи, соответственно нельзя редактировать из карточки записи вручную. При этом поле также может быть изменено от имени пользователя автоматизациями, запускаемыми во время редактирования карточки или через АПИ.&#x20;

## Технические свойства

В технических свойствах отображаются поля для технических специалистов.

#### ID-поля

Уникальный идентификатор поля в структуре каталога, который при необходимости  пользователь может задать сам.&#x20;

#### Служебная заметка

Поле внутренних комментариев для пользователей, которые настраивают каталог.
