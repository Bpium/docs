# Зависимости

### Зачем это нужно

Когда каталог разрастается, поля начинают использоваться в разных местах: на них ссылаются формулы, по ним строятся условия видимости, их читают автоматизации. Если удалить или переименовать такое поле не зная о связях — часть логики перестанет работать без каких-либо предупреждений.

### Как посмотреть зависимости

1. Откройте конструктор каталога и нажмите на поле.
2. В панели свойств справа перейдите на вкладку «Зависимости».
3. Система покажет список всех связей этого поля. Нажмите на любой элемент списка, чтобы увидеть подробности.
