# Текстовый редактор

### Текстовый редактор

Данное поле в отличие от стандартного поля для текста позволяет стилизовать текст:&#x20;

* Настройка заголовков
* Настройка  текста: жирный, курсив, подчеркивание, зачеркивание
* Настройка цвета текста
* Настройка цвета фона
* Настройка нумерации списка
* Настройка отступов и выравнивания текста
* Поле для цитирования
* Поле для кода
