# Дата

## Элементы поля

**Галочка «По умолчанию: текущая дата»** устанавливает в поле текущую дату при создании новой записи.

**Поле с выпадающим списком.** Позволяет выбрать формат отображения даты.

<figure><img src="../../../../.gitbook/assets/image (250).png" alt=""><figcaption></figcaption></figure>

## Форматы поля&#x20;

**Дата.** Позволяет указывать дату в формате ДД.ММ.ГГ&#x20;

**Дата с временем.** Позволяет указывать дату в формате ДД.ММ.ГГ и время в формате ЧЧ.ММ. Поле дата с временем учитывает часовой пояс сотрудников. Например, запись с указанным временем совещания сотрудники в разных часовых поясах увидят по разному, для каждого будет применен его часовой пояс.

**Выбор недели.** Позволяет указывать порядковый номер недели и год.

**Выбор месяца.** Позволяет указывать краткое название месяца и год.

**Выбор квартала.** Позволяет указывать порядковый номер квартала и год.

**Выбор года.** Позволяет указывать только год.
