# Текст

## Когда использовать

Используйте поле Текст, когда значение — это слова или символы, а не число. Типичные примеры:

* Название записи: имя клиента, название проекта, тема письма
* Идентификаторы: номер договора, артикул, ИНН, серия и номер паспорта
* Короткие заметки и комментарии
* Произвольный текст: описание задачи, условия договора

Если нужно хранить число и использовать его в вычислениях — суммировать, сравнивать, считать среднее — используйте [**Число**](chislo.md). Если написать цифры в текстовом поле, они будут строкой: их можно выводить в формулах, но складывать как числа не получится.

Если нужен форматированный текст с заголовками, списками и таблицами — используйте [**Текстовый редактор**](tekstovyi-redaktor.md).

## Форматы поля

**Текст.** Однострочный ввод до 256 символов. Для большинства полей: названия, идентификаторы, короткие значения.

**Текст по маске.** Ввод строго по шаблону. Для номеров договоров, паспортных данных, артикулов — когда структура значения всегда одинакова.

**Многострочный текст.** Несколько строк без форматирования. Для заметок, описаний, условий — когда текст длинный, но разметка не нужна.

## Применение формата «Текст по маске»

Маска — это шаблон, который задает допустимую структуру значения. Сотрудник вводит символы только в разрешенные позиции, остальные символы зафиксированы. Это снижает вероятность ошибок при вводе.

Маска задаётся строкой такой же длины, что и ожидаемое значение. Каждый символ маски — либо служебный (разрешает определенный тип ввода), либо постоянный (отображается в поле как есть).

Все остальные символы в маске считаются постоянными — они отображаются в поле как есть и не требуют ввода.

**Примеры шаблонов масок ввода.**

Ниже приведены примеры использования масок ввода и их расшифровка.

* `1111 №111111` — серия и номер паспорта
* `ДС № 11111`— номер договора, которая начинается с букв ДС
* `1111 1111 1111 1111` — номер кредитной карты банка

## Использование в формулах

Поле Текст участвует в формулах как строка. Это позволяет автоматически собирать значения из нескольких полей или формировать номера по шаблону.

Несколько примеров:

* Собрать полное имя из трёх полей: <mark style="color:$primary;">CONCATENATE({Фамилия}, " ", {Имя}, " ", {Отчество})</mark>
* Сформировать номер договора: <mark style="color:$primary;">CONCATENATE("ДС-", {ID записи})</mark>
* Привести текст к верхнему регистру: <mark style="color:$primary;">UPPER({Название})</mark>
* Проверить, заполнено ли поле: <mark style="color:$primary;">IF(ISEMPTY({Комментарий}), "Нет комментария", {Комментарий})</mark>

{% hint style="warning" %}
Если записать цифры в текстовое поле, сложить их как числа через формулу не получится — система воспринимает их как строку. Для вычислений всегда используйте поле Число.
{% endhint %}

Подробнее о текстовых функциях читайте в разделе [Формулы в полях](../../formuly-v-polyakh.md).
