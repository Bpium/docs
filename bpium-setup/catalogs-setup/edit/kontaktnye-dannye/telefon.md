# Телефон

![](../../../../.gitbook/assets/9-screenshot.png)

## Когда использовать

Используйте поле Телефон вместо обычного Текста везде, где хранится номер телефона.\
Типичные примеры:

* Телефон клиента или контактного лица
* Рабочий и мобильный номера сотрудника
* Многоканальный номер компании и прямой номер менеджера

## Несколько номеров в одном поле

В одно поле Телефон можно добавить сколько угодно номеров. Каждому номеру можно дать подпись — произвольный текст для идентификации: «Личный», «Рабочий», «Секретарь» и любой другой.

Это удобнее, чем заводить отдельное поле под каждый номер: все контакты одного человека хранятся вместе и отображаются компактно.
