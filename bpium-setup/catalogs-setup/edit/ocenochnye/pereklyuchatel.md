# Переключатель

<figure><img src="../../../../.gitbook/assets/Demo-Туры-03-23-2026_04_44_PM.png" alt=""><figcaption></figcaption></figure>

### Когда использовать

Используйте Переключатель когда у записи есть признак, который может быть только в двух состояниях. Типичные примеры:

* Активен / Неактивен — для клиентов, сотрудников, товаров
* Оплачено / Не оплачено
* Согласовано / Не согласовано
* Публично / Скрыто

### Настройка

#### Включён по умолчанию

Опция **«Включён по умолчанию»** устанавливает переключатель в положение «включено» при создании новой записи.
