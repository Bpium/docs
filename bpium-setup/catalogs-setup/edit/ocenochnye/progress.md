# Прогресс

![](../../../../.gitbook/assets/17-screenshot.png)

### Когда использовать

Используйте Прогресс когда нужно отобразить степень выполнения задачи, вероятность или заполненность в виде визуальной шкалы. Типичные примеры:

* Процент выполнения проекта или задачи
* Вероятность закрытия сделки
* Заполненность склада или буфера
* Уровень готовности материала

### Настройка

Поле не имеет дополнительных параметров. Сотрудник вводит значение вручную от 0 до 100.
