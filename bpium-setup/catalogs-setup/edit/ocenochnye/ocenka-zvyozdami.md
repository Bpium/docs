# Оценка звёздами

![](../../../../.gitbook/assets/18-screenshot.png)

### Когда использовать

Используйте Оценку звёздами когда нужно собрать субъективную оценку по пятибалльной шкале. Типичные примеры:

* Оценка качества звонка или встречи
* Рейтинг клиента или партнёра
* Оценка кандидата после собеседования
* NPS-оценка по пятибалльной шкале

### Настройка

Поле не имеет дополнительных параметров. Сотрудник выбирает количество звёзд кликом — от 1 до 5.&#x20;
