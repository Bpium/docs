# Процессы

## Свойства

| Поле              | Описание                                                                                                                                                                                                                                                                                                                                                                                                                     |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Дата запуска      | Дата и время запуска процесса.                                                                                                                                                                                                                                                                                                                                                                                               |
| Сценарий          | Ссылка на запись в каталоге «Сценарии» — какая автоматизация была запущена.                                                                                                                                                                                                                                                                                                                                                  |
| Тип запуска       | <p>Асинхронный — Бипиум запускает процесс в фоне и не возвращает результат инициирующему событию. Используется для уведомлений: отправка письма, обновление связанной записи. Ограничение: 1 сутки (в облаке).</p><p></p><p>Синхронный — Бипиум ждёт результата процесса, чтобы вернуть его событию. Используется для запросов: разрешить или запретить операцию, показать сообщение сотруднику. Ограничение: 60 секунд.</p> |
| Код               | Уникальный идентификатор конкретного запуска процесса.                                                                                                                                                                                                                                                                                                                                                                       |
| Событие           | Подписка из каталога «События», которая запустила этот процесс.                                                                                                                                                                                                                                                                                                                                                              |
| Параметры запуска | Входные данные, переданные сценарию — контекст записи в момент запуска.                                                                                                                                                                                                                                                                                                                                                      |
| Инициатор         | Сотрудник, действие которого запустило процесс.                                                                                                                                                                                                                                                                                                                                                                              |
| Статус            | Текущая стадия или итог выполнения — см. таблицу статусов выше.                                                                                                                                                                                                                                                                                                                                                              |
| Дата завершения   | Дата и время завершения процесса.                                                                                                                                                                                                                                                                                                                                                                                            |
| Результат         | Данные, которые вернул сценарий. При ошибке — текст с описанием причины сбоя.                                                                                                                                                                                                                                                                                                                                                |
| Длительность      | Время выполнения процесса в секундах.                                                                                                                                                                                                                                                                                                                                                                                        |

### Статусы процесса

| Статус                                                               | Что означает                                                                         |
| -------------------------------------------------------------------- | ------------------------------------------------------------------------------------ |
| <mark style="color:green;background-color:green;">Исполняется</mark> | Процесс запущен и выполняется в данный момент.                                       |
| <mark style="background-color:$info;">Завершён</mark>                | Процесс успешно выполнен. В поле «Результат» — данные, которые вернул сценарий.      |
| <mark style="color:red;background-color:red;">Ошибка</mark>          | Процесс завершился с ошибкой. В поле «Результат» — текст ошибки с описанием причины. |
| <mark style="color:$info;background-color:yellow;">Прерван</mark>    | Процесс был остановлен до завершения — например, превышено время выполнения.         |
