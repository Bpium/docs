# Cлужебные каталоги

## Каталоги раздела «Управление»

При создании домена компании Бипиум создает служебный отдел «Управление».

Доступны как в облачной версии, так и при установке на свой сервер.

## Каталоги раздела «Система»

Появляются только при установке Бипиума на собственный сервер. Хранят служебную информацию для работы самой системы.

{% hint style="warning" %}
Пользователю не рекомендуется напрямую взаимодействовать с каталогами из раздела Системы.
{% endhint %}
