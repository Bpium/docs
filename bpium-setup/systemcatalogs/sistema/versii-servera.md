# Версии сервера

## Введение

Системный каталог **Версии сервера** содержит записи всех версий сервера Бипиум, доступных в системе. Каждая запись представляет собой конкретную версию сервера API, которая может быть использована компаниями для работы.

Каталог **Версии сервера** доступен администраторам серверной версии Бипиум и используется для просмотра информации о версиях сервера в системе.

## Назначение каталога

Каталог **Версии сервера** используется для:

* **Просмотра версий сервера**: информация о доступных версиях сервера API в системе
* **Связи с версиями веб-клиента**: каждая версия сервера связана с определенной версией веб-клиента
* **Управления обновлениями компаний**: версии сервера используются в группах компаний для определения версии, на которую будут обновляться компании

## Доступ к каталогу

Каталог **Версии сервера** находится в разделе **Система** и доступен только администраторам сервера Бипиум.

**Как открыть каталог:**

1. Войдите в систему с правами администратора
2. Перейдите в раздел **Система** (в главном меню)
3. Выберите каталог **Версии сервера**

## Поля каталога

Поля в каталоге отображаются в следующем порядке:

| Поле                   | Тип   | Описание                                                                                                                          |
| ---------------------- | ----- | --------------------------------------------------------------------------------------------------------------------------------- |
| **Версия сервера**     | Текст | Номер версии сервера в формате семантического версионирования (например, `1.2.3`).                                                |
| **Версия веб-клиента** | Связь | Связь с каталогом **Версии веб-клиента**. Определяет, какая версия веб-приложения будет использоваться с этой версией сервера     |
| **Комментарий**        | Текст | Дополнительная информация о версии (опционально). Может содержать описание изменений, дату выпуска или другую полезную информацию |

## Важная информация

### Обновление версий

Версии сервера создаются автоматически при обновлении сервера Бипиум. При обновлении сервера (замена файлов новой сборкой и запуск `bpium-setup` для обновления базы данных) система автоматически создает запись новой версии сервера и связывает ее с соответствующей версией веб-клиента.

> **Важно:** Версии сервера не управляются отдельно. Они создаются автоматически при обновлении сервера согласно [документации по обновлению](https://docs.bpium.ru/deployment/obsluzhivanie/obnovlenie).

### Связь с версиями веб-клиента

Каждая версия сервера связана с одной версией веб-клиента через поле **Версия веб-клиента**. При обновлении компании система автоматически определяет версию веб-клиента из версии сервера компании и загружает соответствующее веб-приложение.

## Связи с другими каталогами

Каталог **Версии сервера** связан с другими системными каталогами:

* **Версии веб-клиента** — версии сервера связаны с версиями веб-клиента через поле **Версия веб-клиента**. Подробнее см. в документации по каталогу Версии веб-клиента
* **Компании** — компании используют версии сервера через поле **Текущая версия** в каталоге **Компании**. Подробнее см. в документации по каталогу Компании
* **Группы компаний** — группы компаний используют версии сервера через поле **Версия** для управления обновлениями компаний. Подробнее см. в документации по каталогу Группы компаний
