# Версии веб-клиента

## Введение

Системный каталог **Версии веб-клиента** содержит записи всех версий веб-приложения (веб-клиента) Бипиум, доступных в системе. Каждая запись представляет собой конкретную версию веб-интерфейса, которая используется компаниями для работы.

Каталог **Версии веб-клиента** доступен администраторам серверной версии Бипиум и используется для просмотра информации о версиях веб-приложения в системе.

## Назначение каталога

Каталог **Версии веб-клиента** используется для:

* **Просмотра версий веб-приложения**: информация о доступных версиях веб-клиента в системе
* **Связи с версиями сервера**: каждая версия сервера связана с определенной версией веб-клиента через каталог "Версии сервера"
* **Определения версии для компаний**: система использует версию веб-клиента для определения, какую версию веб-приложения загружать для конкретной компании

## Доступ к каталогу

Каталог **Версии веб-клиента** находится в разделе **Система** и доступен только администраторам сервера Бипиум.

**Как открыть каталог:**

1. Войдите в систему с правами администратора
2. Перейдите в раздел **Система** (в главном меню)
3. Выберите каталог **Версии веб-клиента**

## Поля каталога

Поля в каталоге отображаются в следующем порядке:

| Поле            | Тип   | Описание                                                                                                                          |
| --------------- | ----- | --------------------------------------------------------------------------------------------------------------------------------- |
| **Версия**      | Текст | Номер версии веб-клиента в формате семантического версионирования (например, `1.2.3`). Должна быть уникальной                     |
| **Комментарий** | Текст | Дополнительная информация о версии (опционально). Может содержать описание изменений, дату выпуска или другую полезную информацию |

> **Примечание:** Версии должны соответствовать формату семантического версионирования (semver): `MAJOR.MINOR.PATCH` (например, `1.0.0`, `2.1.5`).

## Важная информация

### Обновление версий

Версии веб-клиента обновляются автоматически вместе с обновлением сервера Бипиум. При обновлении сервера (замена файлов новой сборкой и запуск `bpium-setup` для обновления базы данных) система автоматически создает записи новых версий веб-клиента и связывает их с версиями сервера.

> **Важно:** Версии веб-клиента не управляются отдельно от версий сервера. Они обновляются автоматически при обновлении сервера согласно [документации по обновлению](https://docs.bpium.ru/deployment/obsluzhivanie/obnovlenie).

### Связь с версиями сервера

Каждая версия сервера (в каталоге **Версии сервера**) связана с одной версией веб-клиента через поле **Версия веб-клиента**. При обновлении компании система автоматически определяет версию веб-клиента из версии сервера компании и загружает соответствующее веб-приложение.

## Связи с другими каталогами

Каталог **Версии веб-клиента** связан с другими системными каталогами:

* **Версии сервера** — версии сервера связаны с версиями веб-клиента через поле **Версия веб-клиента** в каталоге **Версии сервера**. Подробнее см. в документации по каталогу Версии сервера
* **Компании** — компании используют версии веб-клиента через связь с версиями сервера. Подробнее см. в документации по каталогу Компании
