# Календарь

<figure><img src="../../../.gitbook/assets/image (2) (2).png" alt=""><figcaption><p>Режим отображения Календарь.</p></figcaption></figure>

## Структура интерфейса

Кнопка создания новой записи (1). Позволяет создать новую запись.

Кнопка обновления записей (2). Позволяет актуализировать записей, если в каталог были внесены изменения.

Поиск (3). Поиск позволяет найти нужную запись по введенному значению в строке поиска.

Вид отображения (4). В панели можно выбрать вид календарного  отображения.

Кнопка возврата к текущей дате (5). &#x20;

Чат (6). При нажатии на иконку чата система отфильтрует записи и отобразит только те записи, в чате которых есть новые сообщения

Инструменты подкрашивания, сортировки и отображения (7).

Область переключения между месяцами / неделями / днями (8).&#x20;

Панель быстрого  просмотра данных записи (9).

## Виды отображения

Записи в календарном виде отображаются согласно выбранному периоду. Календарный вид отображения показывает данные в трех временных отрезках:&#x20;

* **Месяц** — все дни месяца на одном экране

<figure><img src="../../../.gitbook/assets/image (55).png" alt=""><figcaption></figcaption></figure>

* **Неделя** — текущая неделя с почасовой разбивкой

<figure><img src="../../../.gitbook/assets/image (56).png" alt=""><figcaption></figcaption></figure>

* **День** — один день с детальной почасовой разбивкой

<figure><img src="../../../.gitbook/assets/image (57).png" alt=""><figcaption></figcaption></figure>

* Кнопка **«Сегодня»** возвращает к текущей дате в любом из режимов.

<figure><img src="../../../.gitbook/assets/image (58).png" alt=""><figcaption></figcaption></figure>

### Создание записи из календаря

Нажмите на пустое место в нужном дне — откроется форма создания новой записи с уже проставленной датой.

{% hint style="info" %}
Если у записи не указана дата окончания, она отображается только в день начала. Чтобы событие тянулось на несколько дней — укажите поле с датой окончания в настройках календарного вида.
{% endhint %}
