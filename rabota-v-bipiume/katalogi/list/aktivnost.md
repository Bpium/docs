# Активность

## Как устроена лента

Все изменения отображаются в хронологическом порядке — самые свежие сверху. Каждое событие в ленте содержит название изменённой записи, имя сотрудника, дату и время, перечень изменённых полей. Для большинства изменений видно что было и что стало — в формате «Было → Стало». Новые даты отделяются разделительной линией для удобной навигации. Комментарии из чатов записей тоже отображаются в ленте.

<figure><img src="../../../.gitbook/assets/пример события в ленте — поле «Статус», Было → Стало.png" alt=""><figcaption><p>Пример события в ленте — поле «Статус», Было → Стало</p></figcaption></figure>

### Фильтрация ленты

Если нужно найти конкретные изменения — воспользуйтесь фильтрами над лентой:

* **По типу действия** — показать все, только создание, только изменение или только удаление записей
* **По сотруднику** — видеть только действия конкретного человека, например чтобы проверить что сделал новый сотрудник за день
* **По полям** — видеть изменения только выбранных полей, например только смену поля «Статус»

Лента активности учитывает ваши права доступа. Вы увидите изменения только по тем записям, к которым у вас есть доступ.
{% endhint %}
