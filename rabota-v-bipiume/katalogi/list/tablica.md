# Таблица

При открытии каталога записи по умолчанию отображаются в режиме отображения Таблица, в которой записи подгружаются по мере пролистывания вниз.

<figure><img src="../../../.gitbook/assets/image (1) (1).png" alt=""><figcaption><p>Режим отображения Таблица.</p></figcaption></figure>

## Структура интерфейса

Кнопка создания новой записи (1). Позволяет создать новую запись.

Кнопка обновления записей (2). Позволяет актуализировать записей, если в каталог были внесены изменения.

Поиск (3). Поиск позволяет найти нужную запись по введенному значению в строке поиска.

Панель отображаемых полей (4). В панели можно менять столбцы полей местами,  перетягивая столбец в нужное место.

Чат (5). При нажатии на иконку чата система отфильтрует записи и отобразит только те записи, в чате которых есть  новые сообщения

**Инструменты подкрашивания, сортировки и отображения (6).**

### Просмотр таблицы

Когда отображено много колонок, таблицу можно смещать горизонтально. Для этого захватите мышкой за свободной место и потяните в соответствующую сторону. Если вы кликните над текстом, то вместо сдвига таблицы начнет выделяться текст.

### Сортировка и закрашивание записей

Вы можете сортировать полученные данные по признаку выбранного вами поля.  Так же можно подкрасить записи цветом значения одного из полей типа Категория для более информативного отображения.  Для этого необходимо нажать иконку окна с карандашом в правом углу.

<figure><img src="../../../.gitbook/assets/image (21) (1).png" alt=""><figcaption></figcaption></figure>

### Выбор колонок для отображения

&#x20;Для настройки видимости вами полей в каталоге необходимо нажать иконку списка в правом углу и выбрать поля, которые должны отображаться. После нажатия выпадет список где можно настроить поля которые вы хотите видеть и очередность их отображения.

<figure><img src="../../../.gitbook/assets/image (74).png" alt=""><figcaption></figcaption></figure>

### Сортировка по столбцу

Чтобы отсортировать таблицу по одной из колонок, кликните по её заголовку. Повторный клик отсортирует данные в обратном порядке. Для примера отсортируем колонку Статус.
