# Таблица

## Структура интерфейса

Кнопка создания новой записи (1). Позволяет создать новую запись.

Кнопка обновления записей (2). Позволяет актуализировать записей, если в каталог были внесены изменения.

Поиск (3). Поиск позволяет найти нужную запись по введенному значению в строке поиска.

Панель отображаемых полей (4). В панели можно менять столбцы полей местами,  перетягивая столбец в нужное место.

Чат (5). При нажатии на иконку чата система отфильтрует записи и отобразит только те записи, в чате которых есть  новые сообщения

Инструменты подкрашивания, сортировки и отображения (6).

### Как работать с таблицей

#### Открыть запись

Кликните на строку — откроется карточка записи. Чтобы отредактировать значение прямо в таблице, не открывая карточку, дважды кликните на нужную ячейку.

#### Изменение размера столбцов

Чтобы изменить ширину столбца, наведите курсор на правую границу заголовка столбца. Когда курсор примет вид двусторонней стрелки, зажмите левую кнопку мыши и потяните в нужную сторону.

При изменении ширины одного столбца автоматически перераспределяется ширина остальных столбцов таблицы.по вертикали.

\[Гифка]

#### Сортировка по столбцу

Кликните на заголовок столбца — записи отсортируются по этому полю. Повторный клик изменит порядок на обратный.

\[Скрин]

#### Выбор отображаемых столбцов

Нажмите иконку списка в правом верхнем углу — откроется панель управления столбцами. Включайте и выключайте поля, меняйте их порядок перетаскиванием.

\[Скрин]

#### Цветовая подсветка строк

Нажмите иконку с карандашом в правом верхнем углу и выберите поле типа «Статус» — строки таблицы подсветятся цветами этого поля. Удобно чтобы с первого взгляда видеть состояние записей.

\[Скрин]

#### Новые сообщения в чатах

Нажмите иконку чата — таблица отфильтруется и покажет только записи, в чате которых есть непрочитанные сообщения.

\[Скрин]

{% hint style="info" %}
Записи подгружаются по мере прокрутки вниз — весь список не загружается сразу. Если нужно найти конкретную запись, используйте поиск или фильтры.
{% endhint %}
