# Массовое редактирование

## Массовое редактирование записей

Пользователю доступно массовое редактирование записей по определенным фильтрам. Лайв-сценарии работать не будут.

### Как пользоваться

1. Нажмите на стрелку рядом с кнопкой «Создать» и выберите «Изменить n записей». Откроется окно с полями каталога.
2. Перед началом редактирования вы можете применить фильтры — количество записей, которые будут изменены, автоматически скорректируется в соответствии с настройками фильтрации.
3. Выберите поля которые хотите изменить и задайте новые значения.
4. Нажмите «Изменить» и подтвердите действие.

\[Скриншот: форма массового редактирования — поля с выбором значений]

### Как изменить значения

Для каждого поля можно выбрать способ изменения:

<table><thead><tr><th width="194">Способ</th><th>Что делает</th></tr></thead><tbody><tr><td><strong>Заменить</strong></td><td>Устанавливает новое значение вместо текущего. Прежнее значение удаляется.</td></tr><tr><td><strong>Добавить</strong></td><td>Добавляет новое значение к уже существующим — без удаления текущих. Например, добавить ещё одного сотрудника к полю «Ответственные».</td></tr><tr><td><strong>Исключить</strong></td><td>Убирает указанное значение из тех, что уже есть в поле — остальные значения не трогает.</td></tr></tbody></table>

{% hint style="info" %}
Массовое редактирование изменяет данные сразу во всех выбранных записях — отменить это действие нельзя. Перед сохранением убедитесь, что выбраны нужные записи и правильные значения.
{% endhint %}
