# Массовое удаление

Массово удалить записи поможет в следующих случаях:&#x20;

* Нужно удалить все записи каталога
* Нужно удалить отобранные записи каталоге, не прибегая к ручному удалению каждой записи отдельно

## Удаление всех записей каталога

Используйте этот способ когда нужно полностью очистить каталог — например, после заполнения тестовыми данными при разработке.

1. Убедитесь, что в каталоге выбран вид **«Все записи»** и сброшены все фильтры — иначе удалятся только отфильтрованные записи, а не все.

\[Скриншот: каталог в виде «Все записи» без активных фильтров]

2. Нажмите на стрелку рядом с кнопкой «Создать» и выберите «Удалить n записей».
3. В открывшемся окне нажмите «Удалить записи».

\[Скриншот: диалог подтверждения удаления с кнопкой «Удалить записи»]

4. Бипиум начнёт удаление. Дождитесь завершения процесса.

\[Скриншот: процесс удаления — индикатор прогресса]

\[Скриншот: завершение удаления — каталог пустой]

### Удаление отобранных записей

Используйте этот способ когда нужно удалить не все записи, а только определённую выборку.

1. Настройте фильтры чтобы в каталоге отображались только те записи, которые нужно удалить.

\[Скриншот: каталог с применённым фильтром — видны только нужные для удаления записи]

2. Нажмите на стрелку рядом с кнопкой «Создать» и выберите «Удалить n записей».
3. В открывшемся окне нажмите «Удалить записи».
4. Дождитесь завершения процесса.

\[Скриншот: диалог удаления — указано количество записей которые будут удалены]

{% hint style="warning" %}
После нажатия «Удалить записи» процесс начнётся немедленно. Уже удалённые записи восстановить вручную невозможно. Процесс можно прервать — но только те записи которые ещё не были удалены к этому моменту останутся нетронутыми.
{% endhint %}
