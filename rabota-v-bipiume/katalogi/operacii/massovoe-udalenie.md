# Массовое удаление

Массово удалить записи поможет в следующих случаях:&#x20;

* Нужно удалить все записи каталога
* Нужно удалить отобранные записи каталоге, не прибегая к ручному удалению каждой записи отдельно

## Удаление всех записей каталога

Используйте этот способ когда нужно полностью очистить каталог — например, после заполнения тестовыми данными при разработке.

1.  Убедитесь, что в каталоге выбран вид **«Все записи»** и сброшены все фильтры — иначе удалятся только отфильтрованные записи, а не все.<br>

    <figure><img src="../../../.gitbook/assets/Каталог в виде «Все записи» без активных фильтров.png" alt=""><figcaption><p>Каталог в виде «Все записи» без активных фильтров</p></figcaption></figure>
2.  Нажмите на стрелку рядом с кнопкой «Создать» и выберите «Удалить n записей».<br>

    <figure><img src="../../../.gitbook/assets/Кнопка удалить все записи.png" alt=""><figcaption><p>Кнопка «Удалить n записей».</p></figcaption></figure>
3.  В открывшемся окне нажмите «Удалить записи».<br>

    <figure><img src="../../../.gitbook/assets/диалог подтверждения удаления с кнопкой «Удалить записи».png" alt=""><figcaption><p>Диалог подтверждения удаления с кнопкой «Удалить записи»</p></figcaption></figure>
4. Бипиум начнёт удаление. Дождитесь завершения процесса.

<figure><img src="../../../.gitbook/assets/Процесс удаления - индикатор процесса.png" alt=""><figcaption><p>Процесс удаления — индикатор процесса.</p></figcaption></figure>

<figure><img src="../../../.gitbook/assets/Пустой каталог после массового удаления записей..png" alt=""><figcaption><p>Пустой каталог после массового удаления записей.</p></figcaption></figure>

### Удаление отобранных записей

Используйте этот способ когда нужно удалить не все записи, а только определённую выборку.

1. Настройте фильтры чтобы в каталоге отображались только те записи, которые нужно удалить.

<figure><img src="../../../.gitbook/assets/Каталог с применённым фильтром — видны только нужные для удаления записи.png" alt=""><figcaption><p>Каталог с применённым фильтром — видны только нужные для удаления записи.</p></figcaption></figure>

2. Нажмите на стрелку рядом с кнопкой «Создать» и выберите «Удалить n записей».
3. В открывшемся окне нажмите «Удалить записи».
4. Дождитесь завершения процесса.

{% hint style="warning" %}
После нажатия «Удалить записи» процесс начнётся немедленно. Уже удалённые записи восстановить вручную невозможно. Процесс можно прервать — но только те записи которые ещё не были удалены к этому моменту останутся нетронутыми.
{% endhint %}
