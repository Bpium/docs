# Экспорт записей



<figure><img src="../../../.gitbook/assets/Экспорт в excel.png" alt=""><figcaption><p>Пункт «Экспорт в Excel» в выпадающем меню</p></figcaption></figure>

## Как экспортировать

1. При необходимости примените фильтры или выберите нужный вид — в файл попадут только те записи, которые сейчас видны в каталоге.
2. Нажмите на стрелку рядом с кнопкой «Создать» и выберите «Экспорт в Excel».
3. Файл скачается автоматически.

<figure><img src="../../../.gitbook/assets/фильтр на экспорт.png" alt=""><figcaption><p>Каталог с применённым фильтром — перед экспортом видны только нужные записи</p></figcaption></figure>

Бипиум сохраняет в файл все записи каталога, а не только те, что на экране. Если заданы условия поиска, то будут экспортированы только записи, удовлетворяющие фильтру.

### Что попадает в файл

* Бипиум экспортирует все записи каталога, а не только те что видны на экране — если список длиннее одной страницы, все записи всё равно попадут в файл.
* Если применены фильтры или выбран вид — экспортируются только записи соответствующие условиям.
* Экспортируются только записи доступные вам по правам.
* Каждая запись — отдельная строка в файле.

### Формат данных в файле

Сложные типы полей преобразуются в текст:

* **Статус** — название значения
* **Набор галочек** — перечисление отмеченных значений через запятую
* **Сотрудник** — имя сотрудника
* **Связанный каталог** — название связанной записи
* **Контакты** — значения контактов через запятую
* **Файл** — поля типа «Файл» не экспортируются

### Права на экспорт

Экспортировать записи могут сотрудники с привилегией «Экспортировать» или более высокой привилегией на каталог или вид.
