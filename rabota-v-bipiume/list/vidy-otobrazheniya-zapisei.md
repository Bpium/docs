# Виды отображения записей

## Табличный режим

При открытии каталога записи отображаются в виде таблицы, которая подгружается по мере пролистывания вниз.

<figure><img src="../../.gitbook/assets/image (52).png" alt=""><figcaption></figcaption></figure>

### Просмотр таблицы

Когда отображено много колонок, таблицу можно смещать горизонтально. Для этого захватите мышкой за свободной место и потяните в соответствующую сторону. Если вы кликните над текстом, то вместо сдвига таблицы начнет выделяться текст.

<figure><img src="../../.gitbook/assets/image (4).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (5).png" alt=""><figcaption></figcaption></figure>

### Сортировка и выделение записей

Вы можете сортировать полученные данные по признаку выбранного вами поля.  Так же можно подкрасить записи цветом значения одного из полей типа Категория для более информативного отображения.  Для этого необходимо нажать иконку окна с карандашом в правом углу.

<figure><img src="../../.gitbook/assets/image.png" alt=""><figcaption></figcaption></figure>

### Выбор колонок для отображения

&#x20;Для настройки видимости вами полей в каталоге необходимо нажать иконку списка в правом углу и выбрать поля, которые должны отображаться. После нажатия выпадет список где можно настроить поля которые вы хотите видеть и очередность их отображения.

<figure><img src="../../.gitbook/assets/image (74).png" alt=""><figcaption></figcaption></figure>

### Сортировка по столбцу

Чтобы отсортировать таблицу по одной из колонок, кликните по её заголовку. Повторный клик отсортирует данные в обратном порядке. Для примера отсортируем колонку Статус.

<figure><img src="../../.gitbook/assets/image (112).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (113).png" alt=""><figcaption></figcaption></figure>

### Поиск записей

Поле быстрого поиска позволяет находить записи по данным в текстовых полях: названиях, именах контактов, телефонах и адресах электронной почты. Для расширенного поиска используйте фильтры.

## Плиточный режим

Плиточный режим работает аналогично Канбан-доске, в которой можно разместить задачи и перетаскивать между столбцами. В плиточном режиме отображения можно просмотреть до 500 записей.

<figure><img src="../../.gitbook/assets/image (53).png" alt=""><figcaption></figcaption></figure>

### Построение <a href="#postroenie" id="postroenie"></a>

В плиточном режиме вы можете настроить параметры вывода по горизонтали и вертикали. Для этого необходимо нажать на иконку списка в верхней правой части экрана.

Так же можно подкрасить плитки значениями цветов одного из полей типа Категория для более информативного отображения. В результате существует возможность сортировать полученные данные по признаку числа ваших полей.

<figure><img src="../../.gitbook/assets/image (81).png" alt=""><figcaption></figcaption></figure>

### Видимость столбцов

Нажмите на иконку списка и настройте поля, которые вы хотите видеть, и очередность их отображения.

<figure><img src="../../.gitbook/assets/image (82).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (83).png" alt=""><figcaption></figcaption></figure>

### Изменение размеров столбцов <a href="#izmenenie-razmerov-stolbcov" id="izmenenie-razmerov-stolbcov"></a>

Для изменения размера столбцов необходимо навести курсор на международный столбец в строку разложения по горизонтали и потянуть в нужную сторону. При изменении размеров одного столбца автоматически меняется размер других. Отдельно можно настроить поворот столбца разложения по вертикали.

### Поиск записей <a href="#poisk-zapisei-1" id="poisk-zapisei-1"></a>

Поле быстрого поиска так же работает и в режиме «Плит». Оно позволяет просматривать записи по данным в текстовых полях: названия, имена контактов, телефоны и адреса электронной почты. Для расширенного поиска используйте фильтры.
