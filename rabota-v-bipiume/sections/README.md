# Интерфейс системы

Интерфейс Бипиума состоит из следующего инструментария:&#x20;

* Меню разделов
* Меню каталогов
* Меню настроек
* Панель чатов
* Рабочая область
* Расширенное меню

## Меню разделов

Меню разделов представляет собой область расположения всех разделов системы и находится в самой верхней части интерфейса системы.&#x20;

Меню разделов также содержит следующий инструментарий:

* Панель чатов
* Документация&#x20;
* Язык&#x20;
* Меню настроек&#x20;

### Разделы

Разделы служат для наведения порядка. С помощью них вы можете группировать каталоги со связанными по смыслу данными. В Бипиуме нет ограничения на число разделов.

### Документация

При нажатии на иконку документации открыввается поисковое окно, в котором можно найти и просмотреть подсказки по работе в Бипиум.

### Язык

При нажатии на иконку языка можно выбрать предпочтительный язык интерфейса.

### Меню настроек&#x20;

Меню настроек представляет собой область, в которой содержится различный инструментарий для пользователя. Меню настроек состоит из следующих элементов:&#x20;

* Название и адрес системы
* Кнопка выхода из учетной записи
* Настройка фона
* Кнопка открытия Документации Бипиум на сайте
* Кнопка отправки вопроса технической поддержке

## Меню каталогов

Меню каталогов представляет собой область расположения всех каталогов открытого раздела и находится под меню с разделами.

### Каталоги

Каталог — набор однотипных записей. Например, список клиентов, заказов, обращений, городов.



## Панель чатов

Панель чатов представляет собой окно, в котором в хронологическом порядке отображаются чаты по записям, в которых в недавно были отправлены или получены сообщения.

### Просмотр сообщений через панель чата

Нажмите на иконку чата и в открывшейся панели выберите интересующий вас чат. Система откроет анкету записи и чат по ней.

## Рабочая область

Рабочая область содержит в себе различные инструменты для работы с записями в каталоге:&#x20;

* Панель фильтров
* Меню настроек каталога
* Режимы отображения
* Панель столбцов
* Кнопка обновления&#x20;
* Меню массовой настройки записей
* Кнопка создания записи
* Панель личных видов
