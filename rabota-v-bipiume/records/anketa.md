# Анкета

## Создание записи

Нажмите кнопку «Создать» в правом верхнем углу каталога — откроется пустая анкета. Заполните нужные поля и нажмите «Сохранить».

<figure><img src="../../.gitbook/assets/Пустая анкета новой записи.png" alt=""><figcaption><p>Пустая анкета новой записи</p></figcaption></figure>

### Обязательные поля

Обязательные для заполнения поля отмечены красной звездой справа от названия поля. Сохранить запись без заполнения обязательных полей не получится — система покажет ошибку и выделит незаполненные обязательные поля.

<figure><img src="../../.gitbook/assets/поле с красной звездочкой — обозначение обязательного поля.png" alt=""><figcaption><p>Поле с красной звездочкой — обозначение обязательного поля</p></figcaption></figure>

<figure><img src="../../.gitbook/assets/попытка сохранить с незаполненным обязательным полем — ошибка.png" alt=""><figcaption><p>Попытка сохранить с незаполненным обязательным полем — ошибка</p></figcaption></figure>

## Редактирование записи

Откройте запись — поля сразу доступны для редактирования, если у вас есть право изменять записи. Измените нужные значения и нажмите «Сохранить».

<figure><img src="../../.gitbook/assets/Кнопка «Сохранить» в правом верхнем углу карточки.png" alt=""><figcaption><p>Кнопка «Сохранить» в правом верхнем углу карточки</p></figcaption></figure>

{% hint style="info" %}
При сохранении Бипиум отправляет на сервер только изменённые поля — это исключает случайное перезатирание данных когда несколько сотрудников работают с одной записью одновременно.
{% endhint %}

## Закрытие без сохранения

Если вы внесли изменения и попытаетесь закрыть запись без сохранения — Бипиум спросит что сделать: сохранить изменения или закрыть без сохранения. Это защита от случайной потери данных.
