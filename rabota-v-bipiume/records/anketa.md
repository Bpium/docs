# Анкета

Анкета — это форма, созданная на основе структуры каталога, которая является «главной вкладкой» записи. При создании новой записи или открытии уже созданной система открывает запись с анкетой

#### Поля для заполнения

<figure><img src="../../.gitbook/assets/image (35).png" alt=""><figcaption></figcaption></figure>

Обязательные для заполнения поля записи помечены красной линией слева от самого поля.&#x20;



<figure><img src="../../.gitbook/assets/image (68).png" alt=""><figcaption></figcaption></figure>

Без заполнения обязательных полей запись нельзя будет сохранить. Необязательные для заполнения поля могут быть пустыми.

<figure><img src="../../.gitbook/assets/image (69).png" alt=""><figcaption></figcaption></figure>

Изменения сохраняются при сохранении всей анкеты. При этом будут сохранены только изменённые поля. Это минимизирует вероятность перезатирания данных при одновременной работе нескольких сотрудников с одними записями.

## Редактирование записи

После создания запись также можно редактировать, если у вас есть доступ редактирования.

Для того, чтобы отредактировать запись необходимо:

* Открыть запись

<figure><img src="../../.gitbook/assets/image (129).png" alt=""><figcaption></figcaption></figure>

* Внести необходимые изменения изменения

<figure><img src="../../.gitbook/assets/image (131).png" alt=""><figcaption></figcaption></figure>

* Нажать «Сохранить»

<figure><img src="../../.gitbook/assets/image (132).png" alt=""><figcaption></figcaption></figure>

Внесенные изменения в записи отобразятся в Истории изменений
