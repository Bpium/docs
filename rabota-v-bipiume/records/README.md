## Структура Записи

Карточка записи состоит из нескольких элементов

* **Вкладки.** Каждая вкладка может содержать в себе секции  и поля для&#x20;

<figure><img src="../../.gitbook/assets/Открыиая карточка записи с выделенными вкладками.png" alt=""><figcaption><p>Открытая карточка записи с выделенными вкладками</p></figcaption></figure>

* **Секции.** Секции позволяют группировать поля записи по функциональным блокам. Секции при необходимости можно скрыть.

<figure><img src="../../.gitbook/assets/Открытая карточка записи с выделенными секциями.png" alt=""><figcaption><p>Открытая карточка записи с выделенными секциями</p></figcaption></figure>

* **Поля.** Элементы для ввода и хранения данных внутри секций.

## Режимы открытия записи

Запись можно открыть двумя способами:

* **Сбоку** — карточка открывается в панели справа от списка записей. Список остаётся видимым, можно переключаться между записями не закрывая карточку.

<figure><img src="../../.gitbook/assets/Карточка открыта сбоку — список записей слева, карточка справа.png" alt=""><figcaption><p>Карточка открыта сбоку — список записей слева, карточка справа</p></figcaption></figure>

* **В отдельном окне** — карточка открывается на весь экран. Удобно для детальной работы с записью у которой много полей.

<figure><img src="../../.gitbook/assets/Кнопка открытия записи в полноэкранном окне.png" alt=""><figcaption><p>Кнопка открытия записи в полноэкранном окне</p></figcaption></figure>

<figure><img src="../../.gitbook/assets/Карточка открыта в отдельном полноэкранном окне.png" alt=""><figcaption><p>Карточка открыта в отдельном полноэкранном окне</p></figcaption></figure>

### Дерево записей

Когда вы открываете связанную запись прямо из карточки другой записи — появляется дерево записей. Оно показывает на каком уровне вложенности вы сейчас находитесь и помогает не потеряться при работе с цепочкой связанных записей.

Например: вы открыли сделку, из неё открыли карточку клиента, из клиента — карточку контактного лица. Дерево покажет всю эту цепочку и позволит быстро вернуться на любой уровень.

<figure><img src="../../.gitbook/assets/Дерево записей.png" alt=""><figcaption><p>Дерево записей</p></figcaption></figure>

Дерево записей видно при открытии карточки в полноэкранном режиме, при просмотре связанной записи и при создании новой записи в связанном поле.
