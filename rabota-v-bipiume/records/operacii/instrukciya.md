# Инструкция

\[Скриншот: вкладка Инструкция внутри записи — текст инструкции]

## Как открыть

Откройте любую запись каталога и нажмите иконку инструкции в строке вкладок. Если иконка не отображается — значит администратор ещё не создал инструкцию для этого каталога.

{% hint style="info" %}
Внутри записи инструкцию можно только читать. Редактировать её можно только из режима отображения «Инструкция» в самом каталоге — и только сотрудникам с правом администрировать каталог.
{% endhint %}
