# Операции

## Клонирование записи

### Особенности

* При клонировании записи, поле типа Файл не клонируется&#x20;
* При клонировании необходимо учитывать, что Связанные записи не клонируются, а указываются те же самые. Соответственно, если в склонированной записи изменить какие-либо значения связанной записи, то они изменятся и в основной записи.

### Как клонировать

Выберите запись, которую необходимо клонировать. В анкете записи откройте настройки записи и выберите «Клонировать».&#x20;

<figure><img src="../../../.gitbook/assets/image (256).png" alt=""><figcaption><p>Клонирование записи.</p></figcaption></figure>

Система откроет карточку записи-клона, в которой можно при необходимости внести изменения. После нажатия «Сохранить» система создаст копию записи.

<figure><img src="../../../.gitbook/assets/image (257).png" alt=""><figcaption><p>Клонирование записи.</p></figcaption></figure>

Копия записи отобразится в каталоге.

<figure><img src="../../../.gitbook/assets/image (258).png" alt=""><figcaption><p>Клонированая запись.</p></figcaption></figure>

## Удаление записи

### Особенности

При удалении записи она удаляется из каталога, но остается в качестве выбранных значений в полях других каталогов. Например, если удалить клиента, то в заказах этого клиента можно будет увидеть его название, однако оно будет перечеркнутым. Открыть такую перечеркнутую запись из связанного каталога нельзя, при необходимости возможно лишь вручную удалить оставшуюся связь или выбрать на ее место другую неудаленную запись.

Удаленные записи восстановить нельзя.

### Как удалить

Удалить запись можно несколькими способами:&#x20;

* Открыть запись, нажать на галочку и выбрать «Удалить запись»

<figure><img src="../../../.gitbook/assets/image (127).png" alt=""><figcaption></figcaption></figure>

* Открыть каталог и нажать на иконку корзины в строке записи

<figure><img src="../../../.gitbook/assets/image (126).png" alt=""><figcaption></figcaption></figure>

При попытке удалить запись система предупредит об удалении всех данных записи
