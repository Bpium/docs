# Чат

Чат представляет собой вкладку внутри записи, в которой сотрудники могут коммуницировать между собой, оставляя заметки, задания, вопросы и др.

### Просмотр чата

Просмотреть чат можно открыв в записи вкладку с иконкой чата.

### Элементы чата

Чат содержит следующие элементы:&#x20;

* Поле для ввода текста
* Тег «@» — кнопка, которая позволяет найти сотрудника, которому необходимо отправить сообщение
* Скрепка — для прикрепления файла
* Телеграм — кнопка отправки сообщения в чат
* Следить — кнопка переключателя отслеживания уведомлений в чате
* Поле  с историей чата

### Подписаться на уведомления в чате

По умолчанию уведомления в чате конкретной записи отключены. Для того, чтобы получать уведомления о новом сообщении в чате конкретной записи необходимо подписаться на уведомления в чате.

Чтобы подписаться уведомления в чате необходимо нажать на кнопку «Переключить», чтобы переключатель перешел в активный режим.&#x20;

### Отправить сообщение конкретному сотруднику (тегнуть)

Для того, чтобы сообщение в чате пришло конкретному сотруднику необходимо тегнуть сотрудника. Для этого нажмите на иконку «@», а затем нажмите на поле для ввода текста. Система отобразит список сотрудников рядом с иконкой «@» . Выберите сотрудника и напишите сообщение&#x20;
